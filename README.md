A lightweight, single-file browser-based tool for testing your Xbox One controller inputs in real time — no installation, no dependencies,

Features

Analog Stick Visualisation — Live canvas rendering for both left and right sticks, with deadzone indicator, position dot, and X/Y axis readouts
Trigger Bars — Smooth fill bars for LT and RT with precise 0.00–1.00 value display
All Buttons Supported — A, B, X, Y (in Xbox colours), LB, RB, D-Pad (all 4 directions), Back, Start, Xbox button, and L3/R3 stick clicks
Event Log — Timestamped press and release history for every button
Auto-detect — Plug in your controller and press any button to activate; disconnection is handled gracefully
